# INT8-Systolic-Array-Accelerator-for-AI-Computation-RTL-to-GDSII
This project is an implementation of an INT8 Systolic Array for AI Accelerator with complete ASIC Design flow all the way from RTL design to Synthesis, FPGA validation and ASIC Backend flow including timing closure, floorplan, DRC and LVS, finally producing GDSII file which can be directly used for chip fabrication.
